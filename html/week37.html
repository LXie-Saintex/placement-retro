Things I enjoyed most this week: Product Switch page styling

I had worked on wwp-139 all week and my code will go into release next Monday if everything turns out alright. It's so nice to have worked on somthing that has value (for Rupert Murdoc at least). I spent 2 days on running the old MAIN platform's Java microservices which had been quite a magical mystery tour. it was mellow but can't really say I learnt something from it. Not until Thursday was I told the priority is to add the stylings. I spent all day to add plain css to plain HTML, which was surprisingly rewarding as it was real work that goes into production quickly. Had some difficulties with the positioning of the header image as the browsers cut it off for reasons still unknown to me. I combined absolute and relative positioning to fix this issue and the end result was pretty good as the styling behaviour is almost identical across browsers, devices and layout modes.
